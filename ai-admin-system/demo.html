您好！这是一个功能强大的AI后台管理系统，主要特色包括： 1. **OpenAI兼容API** - 支持GPT-3.5、GPT-4等所有OpenAI格式的模型 2. **MJ画图功能** - 集成MidJourney和DALL-E图像生成 3. **API密钥管理** - 完善的密钥分发、使用统计和权限控制 4. **多模型支持** - 可配置不同的AI服务提供商和模型 5. **美观界面** - 基于Vue3 + Element Plus的现代化UI设计 6. **实时统计** - 详细的使用数据分析和可视化图表 系统采用Spring Boot + MySQL + Vue3技术栈，提供完整的企业级AI服务管理解决方案。
