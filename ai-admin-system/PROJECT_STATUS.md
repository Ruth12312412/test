## 🔌 API兼容性

### OpenAI格式兼容
```bash
# 标准对话接口
POST /api/chat/completions
{
  "model": "gpt-3.5-turbo",
  "messages": [{"role": "user", "content": "Hello"}],
  "stream": true
}

# 图像生成接口

  "n": 1
}
```
